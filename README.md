**LogiTalk — сучасний мінімалістичний чат-бот**

LogiTalk — це проєкт чату з простим та стильним інтерфейсом, створений для зручного спілкування з ботом.
У ньому можна:

* писати та отримувати текстові повідомлення;
* надсилати та переглядати фото

Проєкт орієнтований на швидкість, простоту й приємний користувацький досвід.
